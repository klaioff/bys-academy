#  BYS Academy

**Boost Your Skills - Let's build together!**

Plateforme de formation dédiée aux étudiants de l'IHEC Carthage et HEC Finance Academy.

##  Fonctionnalités

-  Système de formations modulaires
-  Forum de discussions
-  Badges et gamification
-  Timer Pomodoro pour l'étude
-  Authentification Firebase
-  Interface mobile optimisée

##  Technologies

- HTML5 / CSS3 / JavaScript
- Firebase (Authentication & Firestore)
- Responsive Design
